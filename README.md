🎮 Questline Academica Leveling Up the Undergraduate Experience

Questline Academica is a gamified learning platform designed to bridge the engagement gap in higher education. By transforming traditional syllabi into interactive Skill Trees and Boss Battles, we turn passive learners into active "players" of their own education.

🚀 The Vision Undergraduate engagement is at an all-time low. Traditional LMS platforms (Canvas, Blackboard) feel like administrative chores. Questline applies proven RPG (Role-Playing Game) mechanics to pedagogy to drive:

Retention: Through daily "Bounties" and learning streaks.

Mastery: Through competency-based Skill Trees.

Collaboration: Through peer-led "Guild Missions."

🛠️ Tech Stack Frontend: React.js, Tailwind CSS, Framer Motion (for animations).

Backend: Node.js, Express.

Database: PostgreSQL (Relational mapping for prerequisite quest logic).

Feedback Engines: canvas-confetti for dopamine hits and Custom CSS for "Screen Shake" impact.

AI Integration: (Planned) Dynamic hint generation for struggling students.

✨ Key Features Feature Educational Purpose Game Mechanic Skill Tree Visualizes prerequisites and learning paths. Fog of War / Progression Map Boss Battles High-stakes summative assessment. Turn-based Combat (HP-based Quiz) Loot Drops Rewards for high-performing students. RNG (Random Number Generation) Rewards Bounties Encourages daily micro-learning. 🏆 Hackathon "X-Factor" Our platform features Dynamic Difficulty Adjustment (DDA). If a student breezes through a quest, the "Boss Battle" questions become more conceptual (Bloom's Taxonomy Level 4+). If they struggle, the system offers "Power-Up" hints to scaffold their learning.
